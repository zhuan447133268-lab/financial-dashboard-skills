# 字段语义识别模式

本文件列出 financial-data-teller 识别的字段语义模式，用于匹配中文财务表头。

## 时间字段

| 语义 | 匹配关键词 |
|---|---|
| 时间-年月 | 年月、月份、日期、时间、年份、月份、period、month |
| 时间-季度 | 季度、quarter、Q1、Q2、Q3、Q4 |
| 时间-年份 | 年份、年度、year |

## 主体字段

| 语义 | 匹配关键词 |
|---|---|
| 主体-组织 | 公司、分公司、子公司、校区、园区、门店、主体、组织 |
| 主体-城市 | 城市、地区 |
| 主体-区域 | 区域、大区、片区、region |
| 主体-部门 | 部门、事业部、department |

## 金额字段

| 语义 | 匹配关键词 |
|---|---|
| 金额-收入 | 收入、营收、营业额、主营业务收入 |
| 金额-成本 | 成本、主营业务成本 |
| 金额-费用 | 费用、管理费用、销售费用、财务费用、运营费用 |
| 金额-利润 | 利润、净利润、毛利、营业利润、EBIT、EBITDA |
| 金额-预算 | 预算、预算金额、预算收入、预算成本 |
| 金额-实际 | 实际、发生额、实际金额、实际收入、实际成本 |

## 科目字段

| 语义 | 匹配关键词 |
|---|---|
| 科目-编码 | 科目编码、科目代码、subject code |
| 科目-名称 | 科目名称、科目、subject name |
| 科目-层级 | 一级科目、二级科目、三级科目 |

## 扩展方法

如需支持新的行业表头，在本文件中添加匹配关键词即可。
