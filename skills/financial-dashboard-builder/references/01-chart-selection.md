# 维度 → 图表类型映射

financial-dashboard-builder 根据维度类型选择 ECharts 图表。

## 映射表

| 用户维度 | 内部维度名 | 推荐图表 | 适用场景 |
|---|---|---|---|
| 月度营收趋势 | 月度趋势 | 折线图 / 面积图 | 体现时间变化 |
| 月度成本趋势 | 月度趋势 | 折线图 / 面积图 | 体现时间变化 |
| 预算达成率 | 预算达成率 | 柱状图 + 目标线 | 体现目标差距 |
| 城市经营对比 | 城市对比 | 分组柱状图 + 折线 | 多主体对比 |
| 区域经营对比 | 城市对比 | 分组柱状图 + 折线 | 多主体对比 |
| 单园利润排行 | 单园排行 | 条形图 | 排行榜 |
| 科目结构 | 科目结构 | 饼图 / 旭日图 | 结构占比 |

## 选择原则

1. **时间趋势用折线**：不要为时间维度用柱状图，折线更易读
2. **对比用柱状/条形**：分类对比用柱状，排行用横向条形
3. **达成率要加目标线**：100% 目标线是必须的
4. **结构占比用饼图要慎重**：类别超过 7 个时改用条形图
5. **不要堆叠过多系列**：一个图表不超过 4 个系列
