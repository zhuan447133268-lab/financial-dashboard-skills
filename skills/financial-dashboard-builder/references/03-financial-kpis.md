# 经营分析 KPI 定义

本文件定义 financial-dashboard-builder 使用的核心 KPI。

## 收入类

| KPI | 公式 | 口径说明 |
|---|---|---|
| 实际收入 | sum(实际收入) | 按确认期间汇总 |
| 预算收入 | sum(预算收入) | 按确认期间汇总 |
| 收入达成率 | 实际收入 / 预算收入 | 低于 80% 标红，80%-90% 标黄 |

## 成本类

| KPI | 公式 | 口径说明 |
|---|---|---|
| 实际成本 | sum(实际成本) | 按确认期间汇总 |
| 预算成本 | sum(预算成本) | 按确认期间汇总 |
| 成本偏差率 | (实际成本 - 预算成本) / 预算成本 | 正值为超支 |

## 利润类

| KPI | 公式 | 口径说明 |
|---|---|---|
| 利润 | 实际收入 - 实际成本 | 单园/单城市/单月均可计算 |
| 利润率 | 利润 / 实际收入 | 通常保留 1 位小数 |

## 建议规则

| 触发条件 | 建议等级 | 建议文本模板 |
|---|---|---|
| 收入达成率 < 80% | 高 | 整体收入达成率仅 X%，低于警戒线，需重点关注 |
| 收入达成率 80%-90% | 中 | 整体收入达成率 X%，建议分析差异 |
| 成本偏差率 > 10% | 高 | 实际成本超出预算 X%，建议核查成本科目 |
| 成本偏差率 5%-10% | 中 | 实际成本超出预算 X%，建议关注主要成本项 |
| 实际收入连续 2 期下降 | 高 | 实际收入已连续 N 个月下滑，需立即分析 |

## 注意事项

- 所有计算前先检查缺失值，缺失值不参与汇总
- 单位必须全局统一
- 时间断档要标注，不能默认补 0
